CPS_StaticRequest
This class, which is derived from the CPS_Request class, can be used to send pre-formed XML requests to the storage. If you already have the XML request, you can send it to the API by using this class. Note that parameters from the CPS_Connection object such as storage name, username and password will not be used to form the request.
