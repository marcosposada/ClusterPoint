CPS_SearchRequestsetQuery Method (String)Clusterpoint Server .NET API library
Sets search query. Use can use Utils class functions for generating query.
Examples
This example show how to use Utils function to generate search query.
Copy
r.setQuery("(" + Utils.CPS_Term("predefined_term", "generated_fields/some_field") + Utils.CPS_Term(user_supplied_term, "searchable_fields/text") + ")");
