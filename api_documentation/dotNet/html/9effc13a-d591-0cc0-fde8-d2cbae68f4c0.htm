CPS_Simplesearch Method (String, Int32, Int32, DictionaryString, String, String, CPS_ResponseDOC_TYPE, String)Clusterpoint Server .NET API library
Performs a search command. Returns the documents found in a Dictionary with document IDs as keys and document contents as values. Returned document type depends on returnType parameter. If returnType is DOC_TYPE_SIMPLEXML, returns Dictionary<string, CPS_SimpleXML>. If returnType is DOC_TYPE_ARRAY, returns null - not supported yet. If returnType is DOC_TYPE_STDCLASS, returns Dictionary<string, XmlElement>.
