CPS_SimplelookupSingle Method Clusterpoint Server .NET API library
Lookups single document in Clusterpoint Server storage. Returned type depends on passed returnType parameter. If returnType is DOC_TYPE_SIMPLEXML, returns CPS_SimpleXML. If returnType is DOC_TYPE_ARRAY, returns null - not supported yet. If returnType is DOC_TYPE_STDCLASS, returns XmlElement.
