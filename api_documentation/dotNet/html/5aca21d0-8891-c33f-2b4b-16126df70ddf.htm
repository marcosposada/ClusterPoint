CPS_SimplelistFirst Method Clusterpoint Server .NET API library
Performs list-first command. Returned type depends on passed returnType parameter. If returnType is DOC_TYPE_SIMPLEXML, returns Dictionary<string, CPS_SimpleXML>. If returnType is DOC_TYPE_ARRAY, returns null - not supported yet. If returnType is DOC_TYPE_STDCLASS, returns Dictionary<string, XmlElement>.
