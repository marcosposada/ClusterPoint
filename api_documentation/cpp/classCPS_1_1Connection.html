Connection API to Clusterpoint Server.

Constructs an instance of the Connection class. Note that it doesn't necessarily make a connection to CPS when the constructor is called.

Parameters
connectionStringSpecifies the connection string, such as tcp://127.0.0.1:5550
storageNameThe name of the storage you want to connect to
usernameUsername for authenticating with the storage
passwordPassword for this user
documentRootXpathDocument root tag name. Default is "document"
documentIdXpathDocument ID xpath. Default is "document/id"
